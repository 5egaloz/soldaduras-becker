# R1 — Página de Soldadura + Kit de Marca

Página web (landing) para dar visibilidad al negocio de soldadura/herrería, más el
kit de marca que alimenta a **Oficina IA Bot** para generar imágenes y contenido.

## 📂 Qué hay aquí
| Archivo / carpeta | Para qué |
|---|---|
| `index.html` | La página web (landing). |
| `style.css` / `script.js` | Diseño y botones de WhatsApp. |
| `assets/trabajos/` | Tus fotos de trabajos (trabajo-1.jpg … trabajo-6.jpg). |
| `marca/negocio.md` | Datos del negocio = fuente de verdad. |
| `marca/prompts-imagenes.md` | Prompts para que Oficina IA Bot genere imágenes. |
| `TAREAS.md` | ✅ **Tu lista de pendientes. Empieza por aquí.** |

## 🌐 Cómo se publica gratis (GitHub Pages)
1. Sube este repo a GitHub (ver `TAREAS.md`).
2. En GitHub: **Settings → Pages → Branch: `main` / `(root)` → Save**.
3. En ~1 min queda en: `https://TU-USUARIO.github.io/R1/`
4. Ese link lo compartes por WhatsApp, Facebook, tarjetas, etc.

## ✏️ Qué editar para personalizar (en `index.html`)
- **Número de WhatsApp:** busca `data-wa="569XXXXXXXX"` (aparece 4 veces) y pon el tuyo.
- **Nombre del negocio:** busca `Soldadura [TU NEGOCIO]`.
- **Teléfono, zona y horario:** sección de contacto (busca `data-edit`).
- **Fotos:** mételas en `assets/trabajos/` con los nombres indicados.
